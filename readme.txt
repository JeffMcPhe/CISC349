Update the server endpoint URL in the app code to point to your Flask server.
Build and run the app on an Android emulator or physical device.
Usage
Launch the HUstagram app on your Android device.
Use the camera functionality to take a picture.
Enter a comment for the image (optional).
Tap on the upload button to upload the image to the server.
View all uploaded images in the app's gallery.
Tap on an image to view its full version along with the associated comment and upload date.
